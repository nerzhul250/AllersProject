El repositorio ProyectoAllers contiene una solucion de Visual Studio en C# mas toda la documentación pertinente a lo que vendriamos a llamar un aplicativo que integra tecnicas de analisis de datos para la generación de predicciones o reglas de asociación y recomendaciónes, dado datos de transacciones, items y clientes.

En la carpeta AllersProject contiene las clases que componen la GUI.
En las carpetas Datos y DatosTests se encuentran datos de prueba como los escenarios para los tests y en la carpeta UnitTestProject1 se encuentran la implementación de tests.
En la carpeta Docs se encuentra la documentación como los requerimientos funcionales.
En las carpetas Estructura, Extensions y Modelo contienen las clases que componen la logica del negocio del aplicativo.
Finalmente en la carpeta VisualParadigmClassDiagram se encuentran los diagramas de clases que describen la estructura del aplicativo.
